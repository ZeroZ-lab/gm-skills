# draft-generator

初稿生成器。按结构填充内容，先完成再完美。

## 何时使用

- 结构定好了，需要写成完整段落
- 写了开头和结尾，中间不知道怎么展开
- 有要点但不知道怎么写成流畅的文字
- 需要快速出一版能看的初稿

## 输入

1. **结构大纲**：各部分标题/要点
2. **已有素材**：案例、数据、引用
3. **风格偏好**：
   - 口语化（像聊天）
   - 分析型（逻辑清晰）
   - 文学型（有画面感）
   - 专业型（行业术语）

## 输出

按结构逐段生成内容：

```
【第1节：XXX】
[生成内容]

【第2节：XXX】
[生成内容]
...
```

每段附带：
- 本段功能说明（这节在做什么）
- 过渡提示（下节如何衔接）

## 写作原则

- **先完成再完美**：先生成完整内容，不纠结措辞
- **具体优先**：用例子代替抽象陈述
- **读者视角**：每段问"读者此刻在想什么"
- **节奏控制**：长短句交替，避免大段文字

## 使用建议

1. 先生成完整初稿
2. 标记不满意的地方
3. 用 **revision-tuner** 优化
