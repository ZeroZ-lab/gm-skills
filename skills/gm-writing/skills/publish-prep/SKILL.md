# publish-prep

发布准备。文章写好了，让它准备好见人。

## 何时使用

- 正文定稿，需要起标题
- 需要写摘要/导读
- 不知道配什么图，需要生成提示词
- 要发到多个平台，需要调整格式

## 输出

### 1. 标题选项

提供 5 个版本：

| 类型 | 标题 | 适用场景 |
|-----|-----|---------|
| 直接型 | 直白说出内容 | 干货、教程 |
| 好奇型 | 制造悬念 | 故事、案例 |
| 利益型 | 承诺价值 | 方法、攻略 |
| 冲突型 | 制造对立 | 观点、评论 |
| 共鸣型 | 情感共振 | 经历、感悟 |

### 2. 摘要/导语

3 个版本：
- **短版**（50字）：适合小红书/即刻
- **中版**（100字）：适合公众号头条
- **长版**（200字）：适合知乎/ newsletter

### 3. 配图提示词

提供 3 套配图方案：
- 封面图提示词（Midjourney/Stable Diffusion）
- 文中插图建议（在哪里插什么）
- 信息图/图表建议（如果有数据）

### 4. 平台适配

| 平台 | 调整建议 |
|-----|---------|
| 公众号 | 标题长度、分段、引导关注 |
| 小红书 | 加emoji、缩短段落、tag建议 |
| 知乎 | 导语写法、互动引导 |
| 即刻 | 长度控制、话题标签 |

## 使用方式

告诉我：
1. 完整正文
2. 要发哪些平台
3. 标题有没有特殊要求（必须包含/不能出现）
